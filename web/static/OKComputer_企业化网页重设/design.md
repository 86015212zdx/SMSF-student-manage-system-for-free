# 教学管理系统设计方案

## 1. 设计理念

### 整体视觉方向

- 整体视觉将呈现一种精致、权威且高度功能化的氛围，旨在提升教育管理的效率和专业形象。美学上参考《经济学人》的严谨版式与 Polestar 汽车品牌的现代工业美学，强调结构化布局与功能性设计。核心理念是通过克制而精准的设计语言，构建一个值得信赖、清晰且专注于数据的界面环境。

### 内容密度

- 高密度。此系统面向专业用户，需高效处理大量数据与信息。界面将优先展示内容，采用紧凑布局，减少不必要的留白，确保在单一视图内可浏览和操作尽可能多的信息，如表格、图表和关键指标。

## 2. 配色系统

**主色调**

- 背景色：#1A202C（深蓝灰色，专业沉稳）
- 主色：#2D3748（中深灰蓝色，主要信息承载）
- 次要色：#4A5568（中灰蓝色，辅助区分）
- 强调色：#38B2AC（青绿色，用于重要按钮和高亮）

**文字颜色**

- 深色文字（用于浅色背景）：#1A202C
- 浅色文字（用于深色背景）：#E2E8F0

## 3. 字体设计

**标题字体**

- 中文字体
  - Chiron Hei HK（现代几何黑体，线条锐利，科技感强，彰显专业与精准）
- 英文字体
  - Hedvig Letters Sans（个性鲜明，具有强烈的品牌感，为界面注入独特气质）

**正文字体**

- 中文字体
  - Noto Sans SC（结构规范，风格中性，为数据和文本提供最佳可读性）
- 英文字体
  - Quattrocento Sans（经典优雅的无衬线体，温和清晰，适合长文案阅读）

## 4. 图像风格

- 采用抽象和建筑摄影风格，强调结构、光影和质感。图像应简洁、专业，避免使用人物肖像。可辅以极简的线性图标系统，用于数据可视化和界面导航，确保信息传达的清晰与高效。
